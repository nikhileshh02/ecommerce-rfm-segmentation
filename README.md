# E-Commerce Customer Segmentation — RFM Analysis
## Python + MySQL + Power BI | Full End-to-End Pipeline

## Project Overview
Built a complete 3-tool data pipeline for customer segmentation and churn risk analysis.

## Pipeline
Python (RFM Analysis) -> MySQL (Data Storage) -> Power BI (Churn Dashboard)

## What is RFM?
- R = Recency: How recently did the customer purchase?
- F = Frequency: How often do they purchase?
- M = Monetary: How much do they spend?

## Customer Segments
- Consumer | Corporate | Home Office

## Dashboard Visuals
- Scatter Chart: Days Since Purchase (X) vs Total Spent (Y) by Segment
- Treemap: Customer distribution across segments
- Gauge: Churn Risk % vs threshold
- KPI Cards: Total Spent, Customer Count, Average Spent

## Key Insight
Top-left cluster in scatter = high spenders not buying recently = at-risk customers.
Businesses can target these with re-engagement campaigns.
